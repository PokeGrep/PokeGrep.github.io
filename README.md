# PokéGrep
![Build](https://github.com/PokeGrep/PokeGrep.github.io/actions/workflows/deploy.yml/badge.svg)
![Last Commit](https://img.shields.io/github/last-commit/PokeGrep/PokeGrep.github.io)

PokéGrep is a statically generated Pokémon database built for theory crafters. Instead of trying to be another Pokédex, it focuses on the data that matters most for competitive analysis, team building, breeding, and game mechanics.

Powered by Static Site Generation (SSG), every page is pre-generated, resulting in lightning-fast navigation, minimal infrastructure requirements, and no unnecessary server-side computation.
